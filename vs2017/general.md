# Visual Studio 2017 (v141) — General Properties

Для VS 2017 используется Platform Toolset v141 и соответствующие пути.

| Свойство | Значение |
|---|---|
| Platform Toolset | Visual Studio 2017 (v141) |
| Windows SDK Version | 10.0 (latest installed) |
| C++ Language Standard | ISO C++14 |
| Output Directory | `$(SolutionDir)$(Platform)\$(Configuration)\` |
| Intermediate Directory | `$(ShortProjectName)\$(Platform)\$(Configuration)\` |
| Target Name | `$(ProjectName)` |
