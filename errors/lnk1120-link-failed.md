# LNK1120 — Link Failed (линковка не удалась)

## Симптом

```
fatal error LNK1120: 1 unresolved externals
```

LNK1120 — это итоговая ошибка линковщика, которая указывает количество
неразрешенных внешних символов. Она всегда сопровождается LNK2019 или LNK2001.

## Решение

Ошибка LNK1120 сама по себе не указывает на причину — смотрите сопутствующие ошибки:

- **LNK2019** → проверьте пути к библиотекам (Library Directories) и Additional Dependencies.
- **LNK2001** → проверьте SUBSYSTEM и соответствие entry point типу проекта.
- **LNK1104** → не удаётся открыть файл (обычно .lib или .obj).

## Профилактика

После исправления первопричины:
1. Clean solution.
2. Delete Intermediate Directory вручную.
3. Rebuild.
