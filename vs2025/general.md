# Visual Studio 2025 (v144) — General Properties

VS 2025 использует Platform Toolset v144.

| Свойство | Значение |
|---|---|
| Platform Toolset | Visual Studio 2025 (v144) |
| Windows SDK Version | 10.0 (latest installed) или 11.0 |
| C++ Language Standard | ISO C++17 / C++20 (по умолчанию) |
| Output Directory | `$(SolutionDir)$(Platform)\$(Configuration)\` |
| Intermediate Directory | `$(ShortProjectName)\$(Platform)\$(Configuration)\` |
| Target Name | `$(ProjectName)` |

> **Примечание:** VS 2025 может использовать Windows SDK 11.0.
> Проверьте актуальную версию SDK через Visual Studio Installer.
